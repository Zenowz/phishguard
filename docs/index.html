About PhishGuard

PhishGuard is a cybersecurity tool developed to combat phishing attacks by analyzing URLs and providing real-time threat warnings. The system helps users make safer decisions before clicking suspicious links.

Key Features

🔍 URL Analysis

Examines URLs for suspicious patterns commonly used in phishing attacks.

⚡ Real-Time Detection

Instantly checks websites and provides security warnings.

🛡️ Phishing Detection

Detects risky words, unusual domains, and suspicious URL structures.

📊 Threat Classification

Categorizes links as Safe, Suspicious, or Dangerous.
